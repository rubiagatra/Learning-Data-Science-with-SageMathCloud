# Learning Data Science with SageMathCloud

Ini adalah presentasi pada acara Do Everything with Python yang diselenggarkan oleh [surabaya.py](https://github.com/surabaya-py) pada
pada tanggal 10 Desember 2016 berupa slide, IPython dan Sagews.

Anda tinggal login pada [SageMathCloud](https://cloud.sagemath.com/) dan upload notebook dari IPyhton dan Sagews ini
, note: Python yang digunakan pada notebook ini adalah Python 3 sedangkan pada SageMathCloud adalah Python 2.
